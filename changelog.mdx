## Recent Releases

Stay current with ILA's evolution. You receive notifications for major updates directly in your dashboard. Review the highlights below to see how new features enhance your documentation workflows.

<Callout kind="info">
  Enable email notifications in your account settings to get instant alerts for new releases.
</Callout>

<Update label="2024-10-15" description="v1.2.0" tags={["feature", "bugfix"]}>
## New Features

- Added real-time collaboration editing for shared documentation spaces
- Introduced version history with diff viewer for tracking changes
- Integrated search-as-you-type across all project docs

## Bug Fixes

- Fixed rendering issues with nested code blocks in MDX files
- Resolved authentication token refresh errors during long sessions
- Corrected export failures for large documentation sites

## Improvements

- Enhanced performance of page rendering by `>30%`
- Updated UI components for better mobile responsiveness
</Update>

<Update label="2024-09-20" description="v1.1.0" tags={["feature", "improvement"]}>
## New Features

- Launched custom themes with brand color support (default: `#3B82F6`)
- Added API endpoints for programmatic doc management at `https://api.example.com/v1/docs`
- Enabled webhook integrations for CI/CD pipelines

## Improvements

- Optimized sidebar navigation for faster project switching
- Improved accessibility with better keyboard navigation and screen reader support

</Update>

<Update label="2024-08-10" description="v1.0.0" tags={["feature", "breaking"]}>
## New Features

- Core documentation platform with MDX support and component library
- User authentication via `https://auth.example.com`
- Dashboard overview at `https://dashboard.example.com`

## Breaking Changes

- Migrated to new API version; update your integrations to `/v1`
- Removed legacy YAML frontmatter validation

</Update>
