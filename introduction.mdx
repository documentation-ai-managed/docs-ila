## What is ILA?

ILA is a powerful documentation platform that helps you organize, manage, and publish all your project documentation in one centralized space. You create structured docs sites with MDX support, custom themes, and seamless navigation. Whether you document APIs, user guides, or internal wikis, ILA scales with your needs.

You get version control, search, and collaboration features built-in, so teams stay aligned without scattered files. Start with a blank slate or import from Markdown, GitHub, or Notion.

<Callout kind="info">
  ILA supports full MDX, so you embed interactive components like tabs, steps, and code examples directly in your docs.
</Callout>

## Quick Start

Get your first docs site live in minutes.

<Steps>
  <Step title="Create Account" icon="user">
    Sign up at `https://dashboard.example.com/register`.
  </Step>
  <Step title="New Project" icon="plus">
    Select "Documentation Site" and name your project "My Docs".
  </Step>
  <Step title="Add Content">
    Upload Markdown files or write directly in the MDX editor.
  </Step>
  <Step title="Publish" icon="globe">
    Click "Deploy" to generate your live site at `https://docs.example.com/my-docs`.
  </Step>
</Steps>

<CodeGroup tabs="npm,yarn">
  ```bash
  npm install -g @ila/cli
  ila init my-docs
  ila deploy
  ```
  ```bash
  yarn global add @ila/cli
  ila init my-docs
  ila deploy
  ```
</CodeGroup>
